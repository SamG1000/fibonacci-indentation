# fibonacci-indentation
Instead of static levels, use fibonacci numbers to indent your code.

```
void func(int x)
{
 if(x == 1){
  if(x == 2){
   if(x == 3){
     if(x == 5){
        if(x == 8){
             if(x == 13){
                      if(x == 21){
                                  if(x == 34){
                                  }
                      }
             }
        }
     }
   }
  }
 }
}
```
